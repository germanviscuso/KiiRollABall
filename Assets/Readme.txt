KiiRollABall Demo
=================

A demo of Kii Unity SDK for an Android game. The game showcases
the following:

- User login/registration
- Object storage and retrieval
- Push notifications

For more info see Assets/Plugins/Readme.txt and:

http://blog.kii.com/?p=3761

Source code available at:

https://github.com/germanviscuso/KiiRollABall
